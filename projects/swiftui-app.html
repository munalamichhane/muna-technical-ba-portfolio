Project Overview

This project is a SwiftUI-based iOS application designed to support active learning through interactive flashcards. The app allows users to create, manage, and review study questions using a card-based quiz interface.

Key Features

  • Dynamic list of user-created study questions
  • Ability to add, save, and delete questions
  • Interactive quiz mode with card flipping animation
  • Front-and-back card views for questions and answers
  • Multi-view navigation using SwiftUI

Technical Implementation

  • Built using SwiftUI framework
  • State-driven UI updates
  • Component-based view structure
  • Separation of concerns across ContentView, QuestionListView, and QuizView
  • Custom card components for front and back views

What This Project Demonstrates

  • Ability to design and implement user-centered mobile applications
  • Understanding of SwiftUI state management and view lifecycle
  • Experience translating a functional requirement into a working product
  • Clean separation of views and reusable UI components
