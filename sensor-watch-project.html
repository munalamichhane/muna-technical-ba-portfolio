Wearable Sensor Data Analysis Using Google Pixel Watch

Real-Time Motion Data Collection for Research Applications

Project Overview

This project focused on designing and implementing a system to collect real-time motion data from a wearable device (Google Pixel Watch). The goal was to capture accelerometer and gyroscope data and prepare it for structured analysis in research and analytics contexts.

Problem Context

Wearable devices generate continuous streams of high-frequency sensor data that are difficult to collect, synchronize, and analyze reliably. Researchers require clean, time-aligned motion data to study movement patterns and health-related signals.

My Role

Undergraduate Researcher & Software Developer

Key Contributions

  • Developed an Android application in Kotlin to access Pixel Watch sensor APIs
  • Collected accelerometer and gyroscope data for motion analysis
  • Supported data structuring and preparation for research use
  • Documented system design and outcomes through formal presentations
